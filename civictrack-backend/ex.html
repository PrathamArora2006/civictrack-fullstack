Application Overview

CivicTrack is a complaint management system with the following components:

PostgreSQL Database
Spring Boot Backend
React Frontend
Firebase Integration

How It Works:

  1. Frontend (React) sends HTTP requests to Backend (Spring Boot)
  2. Backend processes requests and interacts with PostgreSQL database
  3. Data is persisted in the Docker container running PostgreSQL
  4. Results are returned to the Frontend for display

Database Information

Your PostgreSQL database is running in a Docker container with the following configuration:

POSTGRES_USER=youruser
POSTGRES_PASSWORD=yourpassword
POSTGRES_DB=yourdb
Port: 5432 (mapped to localhost:5432)

The Spring Boot application connects to this database using the configuration in application.properties:

spring.datasource.url=jdbc:postgresql://localhost:5432/yourdb
spring.datasource.username=youruser
spring.datasource.password=yourpassword
spring.jpa.hibernate.ddl-auto=update

Frontend Integration

Your React frontend connects to the Spring Boot backend using API calls.

Example API call from React to fetch complaints:

fetch('http://localhost:8080/api/complaints')
  .then(response => response.json())
  .then(data => console.log(data));

Example API call to submit a new complaint:

fetch('http://localhost:8080/api/complaints', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(complaintData)
})

Running the Frontend

To start your React frontend application:

cd civictrack-frontend
npm install
npm start

The application will be available at http://localhost:3000

Backend Controls

Your Spring Boot backend is already compiled and ready to run.

To start the backend application:

cd civictrack-backend
mvn spring-boot:run

The backend will be available at http://localhost:8080

API Endpoints:

  • GET /api/complaints - Fetch all complaints
  • POST /api/complaints - Create a new complaint
  • GET /api/complaints/{id} - Get a specific complaint
  • PUT /api/complaints/{id} - Update a complaint
  • DELETE /api/complaints/{id} - Delete a complaint
